# 📊 Análisis de Datos - Plataforma Techno Cursos

## 📌 Descripción
Las plataformas de e-learning son fundamentales en la actualidad, ya que permiten un acceso masivo, flexible y personalizado a la educación.

Este proyecto tiene como objetivo el diseño y análisis de una base de datos para la plataforma **Techno Cursos**, enfocada en gestionar cursos, información de estudiantes, su progreso e interacciones dentro del sistema, incluyendo procesos como certificaciones.

A partir de estos datos, se desarrollan análisis que permiten identificar comportamientos clave de los usuarios y apoyar la toma de decisiones estratégicas.

---

## 🛠️ Tecnologías utilizadas
- SQL (gestión y consulta de datos)
- Power BI (visualización y análisis)

---

## 🔍 Proceso realizado
- Diseño de la base de datos relacional  
- Población de datos  
- Consultas SQL para análisis  
- Creación de dashboards interactivos en Power BI  
- Identificación de patrones de comportamiento de los estudiantes  

---

## 📈 Resultados e insights
- Se identificó que muchos estudiantes inician cursos, pero no los finalizan.  
- Se evidenció que los adultos presentan mayor tasa de abandono.  
- Se observaron patrones de consumo que permiten segmentar mejor a los usuarios.  

---

## 💡 Recomendaciones
- Diseñar estrategias para incentivar pagos anuales mediante descuentos y beneficios de fidelización.  
- Implementar un nuevo plan semestral como opción más flexible para los estudiantes.  
- Mejorar el seguimiento y la motivación para aumentar la finalización de cursos.  
- Crear estrategias específicas para reducir el abandono en adultos.
